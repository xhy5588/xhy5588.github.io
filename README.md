# Personal Portfolio

A lightweight personal website built to showcase your GitHub work and growing experience.

## Files

- `index.html`: page structure
- `styles.css`: layout, colors, and responsive design
- `script.js`: editable content for projects, experience, and contact details

## Customize

Edit `script.js` to update:

- your intro text
- project descriptions
- the experience timeline
- LinkedIn, email, and resume links

## Run locally

Open `index.html` directly in a browser, or serve the folder with a static server.

## Deploy

This site is ready for GitHub Pages:

1. Create a repository for the portfolio.
2. Upload these files.
3. Enable GitHub Pages in the repository settings.
